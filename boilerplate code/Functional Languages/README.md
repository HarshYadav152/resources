# Functional Languages

This directory contains boilerplate "Hello, World!" programs for functional programming languages that emphasize immutability, pure functions, and declarative programming paradigms.

## Languages Included

- **Clojure** ([hello.clj](hello.clj)) - Lisp dialect for the JVM
- **Haskell** ([hello.hs](hello.hs)) - Pure functional programming language
- **OCaml** ([hello.ml](hello.ml)) - Multi-paradigm functional language
- **Racket** ([hello.rkt](hello.rkt)) - Scheme-based language with extensive ecosystem
- **Scala** ([hello.scala](hello.scala)) - Functional and object-oriented hybrid for JVM
- **Scheme** ([hello.scm](hello.scm)) - Minimalist Lisp dialect

## Characteristics

Functional languages typically feature:
- First-class and higher-order functions
- Immutable data structures
- Pattern matching
- Type inference
- Lazy evaluation (in some languages)
- Function composition

## Use Cases

- Data transformation pipelines
- Concurrent and parallel programming
- Mathematical computations
- Compiler and interpreter development
- Academic and research projects
