# Scripting Languages

This directory contains boilerplate "Hello, World!" programs for scripting and automation languages.

## Languages Included

- **Bash** ([hello.bash](hello.bash)) - Unix shell scripting
- **Perl** ([hello.pl](hello.pl)) - Text processing and system administration
- **PowerShell** ([hello.ps1](hello.ps1)) - Windows task automation
- **Lua** ([Embedded Systems/hello.lua](Embedded Systems/hello.lua)) - Lightweight embedded scripting

## Characteristics

Scripting languages typically feature:
- Interpreted execution
- Dynamic typing
- Rapid development capabilities
- System integration
- Text processing capabilities
- Platform automation

## Use Cases

- System administration
- Task automation
- Build processes
- Testing and deployment
- Configuration management
- Rapid prototyping
