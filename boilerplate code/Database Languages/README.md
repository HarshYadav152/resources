# Database Languages

This directory contains boilerplate "Hello, World!" programs for database query and manipulation languages.

## Languages Included

- **SQL** ([hello.sql](hello.sql)) - Standard query language
- **T-SQL** ([hello-t.sql](hello-t.sql)) - Microsoft SQL Server extension
- **PL/SQL** ([hello.plsql](hello.plsql)) - Oracle's procedural extension

## Characteristics

Database languages typically provide:
- Data definition and manipulation
- Query optimization
- Transaction control
- Stored procedures and functions
- Triggers and constraints
- Data integrity enforcement

## Use Cases

- Database management
- Data retrieval and reporting
- Business logic implementation
- Data warehouse operations
- Backend application development
